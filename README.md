# Team Guessing Game

A simple number guessing game written in Python. The computer picks a random number between 1 and 100, and you have to guess it. After each guess, you get a hint telling you whether to go higher or lower.

## How to Play

1. Run the script
2. Type your guess and press Enter
3. Follow the hints — **Too low!** or **Too high!**
4. Keep guessing until you find the number
5. At the end, you can choose to play again or quit

## How to Run

Make sure you have Python 3 installed, then run:

```bash
python guessing\_game.py
```

## Example

```
I'm thinking of a number between 1 and 100!
Your guess: 50
Too low!
Your guess: 75
Too high!
Your guess: 63
You win! The number was 63.
You made 3 attempt(s).
Play again? (y/n):
```

## Requirements

* Python 3.x
* No external libraries needed
